# CodeAlpha Scientific Calculator

A modern Scientific Calculator built as part of the **CodeAlpha Frontend Development Internship - Task 2**.

## 📌 Project Overview
This is a fully functional Scientific Calculator with a dark neon theme,
built using HTML, CSS and JavaScript.

## 🚀 Features
- Basic Operations: Addition, Subtraction, Multiplication, Division
- Scientific Functions: sin, cos, tan, log, ln, √, xʸ, |x|
- Constants: π (pi) and e (Euler's number)
- DEG / RAD mode toggle for trigonometric functions
- Real-time expression display
- Backspace and All Clear (AC) support
- Bracket support for complex expressions
- Keyboard support for fast input
- Responsive dark neon themed UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ▶️ How to Run
1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Start calculating!
